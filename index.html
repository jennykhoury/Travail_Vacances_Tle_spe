Exercice 1 : Algèbre & Second Degré

Énoncé : Soit le trinôme $P(x) = 2x^2 - 10x + 12$. Résoudre l'équation $P(x) = 0$ et étudier son signe.

Corrigé :
1. On calcule le discriminant : $\Delta = b^2 - 4ac = (-10)^2 - 4 \times 2 \times 12 = 100 - 96 = 4$.
2. Comme $\Delta > 0$, l'équation admet deux racines réelles distinctes :
$$x_1 = \frac{-b - \sqrt{\Delta}}{2a} = \frac{10 - 2}{4} = 2 \quad \text{et} \quad x_2 = \frac{-b + \sqrt{\Delta}}{2a} = \frac{10 + 2}{4} = 3$$
3. Pour le signe, $a = 2 > 0$, donc $P(x)$ est strictement positif à l'extérieur des racines et strictement négatif entre les racines :
• $P(x) > 0$ pour $x \in ]-\infty ; 2[ \cup ]3 ; +\infty[$
• $P(x) < 0$ pour $x \in ]2 ; 3[$
• $P(x) = 0$ pour $x \in \{2 ; 3\}$

Exercice 2 : Dérivation & Variations

Énoncé : Étudier les variations de la fonction $f(x) = x^3 - 3x^2 + 1$ sur $\mathbb{R}$. Déterminer l'équation de la tangente en $x = 1$.

Corrigé :
1. La dérivée de $f$ est : $f'(x) = 3x^2 - 6x = 3x(x - 2)$.
2. Les racines de $f'(x)$ sont $x = 0$ et $x = 2$. Le signe de $f'(x)$ est celui de $a = 3 > 0$ à l'extérieur :
• $f'(x) > 0$ sur $]-\infty ; 0[ \cup ]2 ; +\infty[$ (la fonction $f$ y est strictement croissante).
• $f'(x) < 0$ sur $]0 ; 2[$ (la fonction $f$ y est strictement décroissante).
3. Équation de la tangente en $x = 1$ : $y = f'(1)(x - 1) + f(1)$.
• $f(1) = 1^3 - 3(1)^2 + 1 = -1$.
• $f'(1) = 3(1)^2 - 6(1) = -3$.
La tangente a pour équation : $y = -3(x - 1) - 1 = -3x + 2$.

Exercice 3 : Suites Numériques

Énoncé : Soit la suite $(u_n)$ définie par $u_0 = 2$ et $u_{n+1} = 0.5 u_n + 3$. Déterminer sa limite.

Corrigé :
1. C'est une suite arithmético-géométrique. On cherche la valeur de stabilisation $L$ (point fixe) :
$$L = 0.5L + 3 \iff 0.5L = 3 \iff L = 6$$
2. On introduit la suite auxiliaire $(v_n)$ définie par $v_n = u_n - L = u_n - 6$.
3. Montrons que $(v_n)$ est géométrique :
$$v_{n+1} = u_{n+1} - 6 = (0.5u_n + 3) - 6 = 0.5u_n - 3 = 0.5(u_n - 6) = 0.5v_n$$
Ainsi, $(v_n)$ est géométrique de raison $q = 0.5$ et de premier terme $v_0 = u_0 - 6 = -4$.
4. On en déduit : $v_n = v_0 \times q^n = -4 \times (0.5)^n$.
5. Ainsi, $u_n = v_n + 6 = -4 \times (0.5)^n + 6$.
Comme $-1 < 0.5 < 1$, $\lim_{n \to +\infty} (0.5)^n = 0$, donc :
$$\lim_{n \to +\infty} u_n = 6$$

Exercice 4 : Fonction Exponentielle

Énoncé : Résoudre dans $\mathbb{R}$ l'inéquation $e^{2x} - 3e^x + 2 \ge 0$.

Corrigé :
1. On effectue le changement de variable $X = e^x$ avec $X > 0$. L'inéquation devient :
$$X^2 - 3X + 2 \ge 0$$
2. On cherche les racines du trinôme $X^2 - 3X + 2$. Le discriminant est $\Delta = 9 - 8 = 1$. Les racines sont :
$$X_1 = \frac{3 - 1}{2} = 1 \quad \text{et} \quad X_2 = \frac{3 + 1}{2} = 2$$
3. Le trinôme est positif à l'extérieur des racines, soit pour $X \le 1$ ou $X \ge 2$.
4. On revient à la variable $x$ :
• $e^x \le 1 \iff x \le 0$ (car $1 = e^0$).
• $e^x \ge 2 \iff x \ge \ln(2)$.
L'ensemble des solutions de l'inéquation est : $S = ]-\infty ; 0] \cup [\ln(2) ; +\infty[$.

Exercice 5 : Produit Scalaire & Géométrie

Énoncé : Soit la droite $d$ passant par $A(1 ; 2)$ admettant le vecteur $\vec{n}(3 ; -1)$ comme vecteur normal. Déterminer une équation cartésienne de $d$.

Corrigé :
1. Puisque $\vec{n}(3 ; -1)$ est normal à $d$, l'équation cartésienne de $d$ s'écrit sous la forme :
$$3x - y + c = 0$$
2. Le point $A(1 ; 2)$ appartient à $d$, ses coordonnées vérifient donc l'équation :
$$3(1) - (2) + c = 0 \iff 1 + c = 0 \iff c = -1$$
Une équation cartésienne de la droite $d$ est donc : $3x - y - 1 = 0$.

Exercice 6 : Probabilités Conditionnelles

Énoncé : Dans un lot, 2% des pièces sont défectueuses. On passe un test automatique. Si la pièce est défectueuse, le test l'identifie comme telle dans 95% des cas. Si la pièce est saine, le test l'identifie comme saine dans 98% des cas (faux positifs de 2%). Quelle est la probabilité conditionnelle qu'une pièce soit défectueuse sachant que le test est positif ?

Corrigé :
Soit $D$ l'événement "la pièce est défectueuse" et $T$ l'événement "le test est positif".
1. Données : $P(D) = 0.02$, $P(\bar{D}) = 0.98$, $P_D(T) = 0.95$ et $P_{\bar{D}}(\bar{T}) = 0.98 \Rightarrow P_{\bar{D}}(T) = 0.02$.
2. On cherche $P_T(D) = \frac{P(D \cap T)}{P(T)}$.
3. Calcul de $P(D \cap T) = P(D) \times P_D(T) = 0.02 \times 0.95 = 0.019$.
4. Calcul de $P(T)$ par la formule des probabilités totales :
$$P(T) = P(D \cap T) + P(\bar{D} \cap T) = 0.019 + P(\bar{D})P_{\bar{D}}(T) = 0.019 + 0.98 \times 0.02 = 0.019 + 0.0196 = 0.0386$$
5. Application de la formule de Bayes :
$$P_T(D) = \frac{0.019}{0.0386} \approx 0.4922 \quad (\text{soit environ } 49,22\%)$$

Exercice 7 : Variables Aléatoires

Énoncé : On lance un dé équilibré. Si on obtient 6, on gagne 10 €. Si on obtient 5, on gagne 2 €. Sinon, on perd 3 €. Déterminer l'espérance mathématique de ce jeu.

Corrigé :
Soit $X$ la variable aléatoire modélisant le gain net en euros.
1. Valeurs possibles de $X$ : $x_1 = 10$, $x_2 = 2$ et $x_3 = -3$.
2. Loi de probabilité de $X$ :
• $P(X = 10) = \frac{1}{6}$ (obtenir le chiffre 6)
• $P(X = 2) = \frac{1}{6}$ (obtenir le chiffre 5)
• $P(X = -3) = \frac{4}{6} = \frac{2}{3}$ (obtenir 1, 2, 3 ou 4)
3. Calcul de l'espérance mathématique $E(X)$ :
$$E(X) = 10 \times \frac{1}{6} + 2 \times \frac{1}{6} + (-3) \times \frac{4}{6} = \frac{10}{6} + \frac{2}{6} - \frac{12}{6} = 0 \text{ €}$$
L'espérance étant nulle, le jeu est parfaitement **équitable**.

Exercice 8 : Trigonométrie

Énoncé : Résoudre dans $]-\pi ; \pi]$ l'équation $\sin(x) = -\frac{\sqrt{3}}{2}$.

Corrigé :
1. On sait d'après le cours que $\sin(-\frac{\pi}{3}) = -\frac{\sqrt{3}}{2}$.
2. L'équation s'écrit $\sin(x) = \sin(-\frac{\pi}{3})$. Dans $]-\pi ; \pi]$, les solutions d'une telle équation sont de la forme :
• $x_1 = -\frac{\pi}{3}$
• $x_2 = \pi - (-\frac{\pi}{3}) = \pi + \frac{\pi}{3} = \frac{4\pi}{3}$.
Cependant, $\frac{4\pi}{3}$ n'appartient pas à l'intervalle d'étude $]-\pi ; \pi]$. On cherche sa mesure principale congruente :
$$x_2 = \frac{4\pi}{3} - 2\pi = -\frac{2\pi}{3}$$
L'ensemble des solutions de l'équation dans $]-\pi ; \pi]$ est : $S = \{-\frac{2\pi}{3} ; -\frac{\pi}{3}\}$.
